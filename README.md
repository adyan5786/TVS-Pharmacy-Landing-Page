# TVS Pharmacy

A small clone of the CVS Health mobile app homepage UI built with HTML and CSS.

## Simple features

- **Top navigation / header:** A header with logo and primary navigation.
- **Search bar:** Input field for searching the app.
- **Promotional hero/banner:** Large promotional area highlighting offers.
- **Quick access shortcuts:** Icons for Pharmacy, Shop, and Account for fast access.
- **Footer with basic links:** Footer containing basic informational links.

## Languages used

- **HTML**
- **CSS**
